# Метеожурнал

Веб-приложение погоды в виде судового/газетного журнала наблюдений. Текущая температура, ощущается как, влажность, ветер, давление, видимость, восход/закат, график температуры на 24 часа и прогноз на 5 дней.

## Особенности

- Поиск погоды по названию города или по геолокации браузера
- Иконки погоды нарисованы вручную в SVG (без внешних картинок и иконок-библиотек)
- График температуры на 24 часа, отрисован через SVG
- Прогноз на 5 дней
- Тёмная тема в стиле старого газетного/судового журнала, шрифты Fraunces и JetBrains Mono
- Никаких фреймворков и сборщиков — чистые HTML, CSS, JS

## Технологии

- HTML5, CSS3, JavaScript (vanilla, без библиотек)
- [OpenWeatherMap API](https://openweathermap.org/api) — текущая погода, прогноз, геокодинг

## Запуск локально

Файлы статические, сборка не нужна. Проще всего открыть через любой локальный сервер (просто открыть `index.html` двойным кликом тоже сработает, но некоторые браузеры ограничивают запросы с `file://`).

Например, через расширение **Live Server** в VS Code:
1. Открыть папку проекта в VS Code
2. Правой кнопкой на `index.html` → **Open with Live Server**

Или через Python (если установлен):
```
python -m http.server 5500
```
и открыть `http://localhost:5500`.

## Использование

1. Получить бесплатный API-ключ на [openweathermap.org/api](https://openweathermap.org/api) (активация ключа может занять до пары часов после регистрации)
2. Вставить ключ в поле на сайте
3. Ввести город или нажать «Определить по геолокации»
4. Нажать «Загрузить сводку»

Ключ хранится только в памяти текущей вкладки браузера и никуда не отправляется, кроме запросов к OpenWeatherMap.
